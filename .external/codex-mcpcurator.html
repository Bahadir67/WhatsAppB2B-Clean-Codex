🚀 Installation & Setup

Quick Install

# Clone the repository
git clone https://github.com/Codex-Data/codex-mcp
cd codex-mcp

# Install dependencies
npm install

# Build and run
npm run build
npm start

Prerequisites

  • Node.js 18+ or Python 3.8+ (depending on implementation)
  • MCP-compatible client (Claude Desktop, Continue, etc.)
  • Basic understanding of Model Context Protocol

⚙️ Configuration

MCP Settings Configuration

{
  "mcpServers": {
    "codex-mcp": {
      "command": "node",
      "args": ["path/to/codex-mcp/dist/index.js"],
      "env": {
        "API_KEY": "your-api-key-here",
        "DEBUG": "false"
      }
    }
  }
}

Environment Variables

Set these environment variables based on your needs:

  • • API_KEY - Your API key (if required)
  • • DEBUG_MODE - Enable debug logging
  • • TIMEOUT - Request timeout in seconds

💡 Usage Examples

Basic Usage

// Example: Basic usage of Codex MCP
const result = await mcpClient.callTool('execute', {
  input: 'your-input-here',
  options: {
    format: 'json',
    timeout: 30000
  }
});

console.log('Result:', result);
